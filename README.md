# fakenewsdetection


# Project Title: Fake News Detection Using Machine Learning

# Dataset Description:

The dataset consists of news articles collected from two categories: Fake News and True News. Each record contains the news content along with attributes such as title, subject, and date. The dataset is used to train machine learning models that can automatically classify news articles as fake or genuine based on their textual content.

# Objective:

The main goal of this project is to identify and classify misleading or fabricated news articles using Natural Language Processing (NLP) and Machine Learning techniques. This helps reduce the spread of misinformation and improves the reliability of online information.

# Features Used

Text – Main news article content (used for prediction)

Label – Target variable

0 = Fake News

1 = True News

# Data Preprocessing

Merged fake and true news datasets.

Assigned labels to each category.

Removed unnecessary columns (title, subject, date).

Cleaned text using Regular Expressions.

Converted text to lowercase and removed special characters.

Shuffled and prepared the dataset for training.

# Machine Learning Models Applied

Logistic Regression

Decision Tree Classifier

Random Forest Classifier

Gradient Boosting Classifier

# Feature Engineering

TF-IDF (Term Frequency-Inverse Document Frequency) Vectorization was used to convert textual data into numerical features suitable for machine learning models.

# Project Description for Resume

Developed a Fake News Detection system using Natural Language Processing (NLP) and Machine Learning techniques to classify news articles as fake or genuine. Implemented text preprocessing, TF-IDF vectorization, and multiple classification algorithms including Logistic Regression, Decision Tree, Random Forest, and Gradient Boosting to achieve accurate news classification.
